-------------------------------------------------------------------------------------
                                  Flight Data Analysis
-------------------------------------------------------------------------------------


Technology: R and Shiny

Software Requirements-
IDE: RStudio v1.0.44 or higher
R: Version 3.3.2 (or higher)
R Packages: shiny, ggplot2, dplyr

Hardware Requirements-
RAM: 16GB or higher
Processor: i5 2.5Ghz or higher

Steps to run the project:

Step 1: Import Flight Data Analysis R project to the RStudio workspace by going to File-> Open project

Step 2: Install the required R packages by typing the below command in console-

	a) install.packages("shiny")
	b) install.packages("ggplot2")
	c) install.packages("dplyr")

Step 3: Make sure that the "Dataset" folder containing all the csv files are present in the project folder

Step 4: Run the source code by directly clicking on the run button or pressing ctrl+alt+R
